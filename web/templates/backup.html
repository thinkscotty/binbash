Export

Download every bin and item as a CSV file you can open in a spreadsheet or keep somewhere safe.

Import

Load items from a binbash or Homebox CSV export — the format is detected automatically. Items are added to your existing inventory by default; bins are matched by name or created if they don't exist yet.

Migrating from Homebox: an item's location must name a bin (e.g. “Garage / Bin 3”), and the deepest bin in the path is used. Items in a location with no bin are skipped, and Homebox-only fields such as quantity and warranty aren't imported.

If checked, all current bins and items are deleted before the file is loaded. Leave unchecked to add these items alongside what you already have.
